# Fuelerz

**Next-Gen Design Studio: Neon, 3D, and AI Web Experiences**

Fuelerz is a Goa-based creative studio delivering futuristic web, branding, and print designs for startups, businesses, and creators. Our site features a fully responsive neon/circuit-inspired UI, built for accessibility, speed, and SEO.

## 🚀 Features

- **Neon/3D Theme:** Modern, animated circuit backgrounds & glowing UI
- **Services:** Web, logo, branding, e-commerce, and print design
- **Order Online:** Instant WhatsApp integration for quick project requests
- **Portfolio:** Live showcase of our best work
- **Accessibility:** High-contrast, keyboard-friendly, and screen-reader ready
- **SEO Optimized:** Fast, semantic HTML, meta tags, and Open Graph

## 📦 Structure

```text
/
├── index.html            # Homepage (neon hero, CTA, quick access)
├── about.html            # Brand story, mission, values
├── services.html         # Service cards with neon/3D visuals
├── portfolio.html        # Filterable project grid
├── orders.html           # Secure, easy-to-use order form (WhatsApp)
├── contact.html          # Contact info + form (email)
├── support.html          # FAQ and help
├── css/
│   ├── style.css         # Base layout and sidebar
│   └── neon-theme.css    # Neon, 3D, circuit, and accessibility styles
├── js/
│   ├── circuit-bg.js     # Animated SVG circuit background
│   └── form-handler.js   # Order & contact form logic
└── assets/
    └── images/           # Logos, hero illustrations, portfolio images
```

## 🕶️ SEO & Performance

- **Meta Tags:** Each page includes `<title>`, `<meta name="description">`, canonical URLs, and Open Graph/Twitter card tags.
- **Header Structure:** Uses `<h1>`, `<h2>`, etc., for semantic clarity.
- **Alt Text:** All images have descriptive `alt` attributes.
- **Lazy Loading:** Portfolio and large images use `loading="lazy"` for speed.
- **Accessible Forms:** All forms use `<label>` and appropriate input types.
- **Mobile-First:** Fully responsive and tested on all device sizes.
- **Sitemap & robots.txt:** Included for search engines (see below).

## 🛠️ Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/goafuel/Fuelerz.git
   ```
2. Open `index.html` in your browser.
3. Customize images and content in `/assets/images/` and `.html` files.

## ✨ Customization

- **Colors:** Change neon colors in `css/neon-theme.css`
- **Portfolio:** Add your images and project descriptions in `portfolio.html`
- **WhatsApp:** Set your WhatsApp number in `js/form-handler.js` and `orders.html`
- **Analytics/Tracking:** Add your analytics snippet before the `</body>` tag.
